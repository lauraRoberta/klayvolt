# Definições documentais — Klyvolt

## Nome do projeto

Klyvolt

## Slogan

Energia sob controle.

## Descrição

A Klyvolt é uma plataforma de gestão inteligente de energia para pequenas empresas. O sistema reúne dashboard, chatbot e registros de consumo para facilitar o controle energético e a tomada de decisão.

## Público-alvo

Pequenas empresas que desejam reduzir gastos com energia elétrica e melhorar seus hábitos sustentáveis.

## Problema

Muitas pequenas empresas não possuem uma forma simples e acessível de visualizar seu consumo de energia e identificar desperdícios.

## Solução

Uma plataforma web que apresenta indicadores, metas, alertas, relatórios e um chatbot de apoio para orientar o usuário.

## Funcionalidades iniciais

- Página principal
- Dashboard
- Chatbot
- Cadastro de equipamentos
- Página sobre
- CSVs de consumo
- Script SQL inicial
